Vue.js 不支持 IE8 及更低版本

  1. vue.js的api

    data

    type: Object | Function

    在定义组件时,data 必须是一个函数, 返回一个新的原始数据对象

    如果 data 仍然是一个普通对象，则所有的实例将指向同一个对象！
    链接

    vm.$log() 只能输出data里面的输出
    在组件里用 this.$log()

    Vue.js 不能检测到 对象 属性的添加或删除

    • $set

      创建vm实例后才能调用此方法

      全局Api
      用于解决 不能检测到属性添加的限制。
      容易与Array.$set()搞混

    • Object.assign()

      添加属性，即对象与对象之间的合并
      但是，添加到对象上的新属性不会触发更新
      解决：创建一个新的对象，包含原对象的属性和新的属性

    • array.$set(index, value)

      问题：

      • 直接用索引设置元素，如 vm.items[0] = {}；
      • 修改数据的长度，如 vm.items.length = 0

      解决：

    • array.$remove()

      watch: {
      ‘a’: function (val, oldVal) {
      console.log(‘new: %s, old: %s’, val, oldVal)
      },
      }

      var vm = new Vue({
      data: { a: 1 },
      computed: {

      // 仅读取，值只须为函数
aDouble: function () {
  return this.a * 2
},
// 读取和设置
aPlus: {
  get: function () {
    return this.a + 1
  },
  set: function (v) {
    this.a = v - 1
  }
}

      }
      })
      vm.aPlus // -> 2 =>调用get()方法
      vm.aPlus = 3
      vm.a // -> 2 =>调用set()方法
      vm.aDouble // -> 4
      ```
